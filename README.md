# react-scripts-boilerplate
基于create-react-app的react开发环境易用模板


## Install

```
git clone https://github.com/tomieric/react-scripts-boilerplate.git [projectName] && cd project
```

Then

```
npm install
```

## Dev

```
npm start
```

## Build

```
npm run build
```
